How to use our Templates

Getting started is easy. All our templates follow a standard structure and use pure Tailwind CSS with no complex build steps required.

1

Download & Unzip

After purchasing via Lemon Squeezy, you will receive a .zip file. Extract this folder to your desired project location.

Tip: Keep a backup of the original .zip file before making edits.
2

Open in VS Code

Open the extracted folder in your code editor. We recommend Visual Studio Code with the "Tailwind CSS IntelliSense" extension installed for the best experience.

3

Customize Colors & Fonts

We use the standard CDN method for Tailwind so you can edit immediately. To customize the theme, open tailwind.config.js inside the assets folder.

theme: {
  extend: {
    colors: {
      primary: '#2563eb', // Change this hex code
    }
  }
}

Common Questions

No! Our templates use the Tailwind CSS CDN scripts by default, so they work instantly in any browser without installation.
